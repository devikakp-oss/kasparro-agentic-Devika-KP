# Visualizations

This folder contains visual representations of the Multi-Agent Product Content Generation System.

## Files
- `agent_flow.md`: Flowcharts showing agent interactions and data flow
- `sequence_diagram.md`: Sequence and state diagrams of agent execution

## How to View
These files contain Mermaid diagrams. GitHub renders them automatically. For local viewing, use a Mermaid-compatible viewer or copy the code into an online editor.

## Diagrams Overview
- **Agent Flow**: Shows how data flows between agents
- **Sequence Diagram**: Illustrates the order of operations
- **State Diagram**: Represents system states during execution

These visualizations help understand the modular architecture and deterministic flow of the system.
